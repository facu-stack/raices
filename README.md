# 🌿 Centro Raíces — Terapia y Recreación Infantil

Sitio web institucional del **Centro Raíces**, un centro terapéutico multidisciplinario infantil dedicado al acompañamiento de niños, niñas y adolescentes con TEA, TDAH y otros desafíos del desarrollo.

## 🧩 Descripción

El sitio presenta la propuesta integral del centro: servicios terapéuticos (psicología, psicopedagogía, fonoaudiología, terapia ocupacional, kinesiología pediátrica, estimulación temprana, integración sensorial, musicoterapia, psicomotricidad, acompañamiento escolar y recreación terapéutica), equipo profesional, instalaciones, testimonios, preguntas frecuentes y un formulario de contacto para solicitar entrevistas.

## 📁 Estructura

```
Proyecto_Raices/
├── index.html          # Página principal (HTML semántico, accesible y SEO-friendly)
├── assets/
│   ├── css/
│   │   └── style.css   # Sistema de diseño completo con variables CSS
│   ├── js/
│   │   └── main.js     # Interactividad: menú mobile, scroll, acordeón FAQ, formulario
│   └── img/            # Carpeta para imágenes (vacía, usa placeholders ilustrativos)
└── README.md
```

## ✨ Características

- **Diseño responsivo** adaptable a dispositivos móviles, tablets y escritorio
- **HTML semántico** con landmarks ARIA para accesibilidad
- **SEO optimizado** con Open Graph, Twitter Cards, meta tags y datos estructurados (Schema.org LocalBusiness)
- **Modo oscuro nativo** mediante `prefers-color-scheme`
- **Animaciones** de aparición (fade-in) al hacer scroll con Intersection Observer
- **Formulario de contacto** con validación del lado del cliente
- **Acordeón FAQ** interactivo
- **Menú de navegación** con efecto de scroll y versión mobile con toggle
- **Botón flotante de WhatsApp**

## 🚀 Uso

Abrí `index.html` en cualquier navegador moderno. No requiere servidor ni dependencias.

```
start index.html
```

## 🛠️ Tecnologías

- HTML5
- CSS3 (variables, flexbox, grid, media queries, animaciones)
- JavaScript vanilla (ES6+)
